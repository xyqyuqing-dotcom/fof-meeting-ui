## 1. 项目概述

### 1.1 背景

作为主要做FOF业务的私募基金管理人，需要建立一套完整的**FOF下投上会流程管理系统**：

```
投资经理发起 → 项目经理上会 → 上会确认 → 投资经理排期 → 投资计划下单
```

### 1.2 上会触发场景

| 场景 | 说明 |
|------|------|
| 白板做方案 | 新建母基金建仓方案 |
| 追加资金 | 母基金追加资金 |
| 调仓 | 母基金调仓操作 |
| 赎回 | 母基金赎回操作 |

### 1.3 管理模式

系统以**"状态切换"**形式进行管理，支持多视图（表格视图 / 母基金视图）查看和操作。

## 3. 数据字段定义

### 3.1 头部信息（Header）

| 字段 | 类型 | 逻辑说明 |
|------|------|----------|
| 操作类型 | 枚举 | 建仓 / 调仓（母基金可有一次建仓和多次调仓） |
| 本次建仓/调仓总金额(万) | 数值输入 | 每次建仓/调仓需输入总金额 |
| 上会日期/方案日期 | 日期选择 | 业务含义为上会日期；豁免场景下称"方案日期" |
| 投资助理 | 自动+下拉单选 | 选项池为资管投资助理岗，自动带出前一次的助理（可修改） |
| 调仓原因 | 文本输入 | 必填≥20字，仅调仓时出现。同一次调仓不同子基金可有不同原因 |

### 3.2 明细行字段（Detail Row）

| 字段 | 类型 | 逻辑说明 | 适用场景 |
|------|------|----------|----------|
| 母基金 | 下拉(带搜索) | 产品成立流程已发起的我司资管产品，格式：代码(名称) | 通用 |
| 营业部/部门 | 下拉单选 | 证券-XXX营业部 / 期货-XXX营业部 | 通用 |
| 服务人员/客户经理 | 文本/下拉 | 取上分员工信息 | 通用 |
| 预期收益 | 文本输入 | — | 通用 |
| 回撤容忍 | 文本输入 | — | 通用 |
| 子基金 | 下拉(带搜索) | 选项池：FOF池 / 公募标的库，格式：代码(名称) | 通用 |
| 子基金管理人 | 自动带出 | 根据子基金代码自动带出 | 通用 |
| 策略 | 自动带出 | 私募显示二级策略，公募显示标的库"类型"字段 | 通用 |
| **策略比例** | 自动计算 | 相同策略的比例合并汇总（相同策略比例相加） | 通用 |
| 金额(万) | 数值输入 | 最多两位小数；可与配置比例联动计算 | 建仓 |
| 配置比例 | 数值输入 | 最多两位小数；= 金额 / 本次总金额 | 建仓 |
| 最新估值日期 | 自动 | 新建或更改母/子基金时取最新估值 | 调仓 |
| 已投金额(万) | 自动 | 取估值表市值 | 调仓 |
| 配置比例 | 自动 | 取估值表市值占净值 | 调仓 |
| 调仓原因 | 文本 | 同一调仓批次内每条可不同原因 | 调仓 |
| 调仓金额(万) | 数值输入 | 最多两位小数 | 调仓 |
| **调仓后金额(万)** | 自动计算 | = 已投金额 + 调仓金额 | 调仓 |
| **调仓后配置比例** | 自动计算 | = 调仓后金额 / (净资产 + 本次调仓总金额) | 调仓 |
| 池子 | 自动 | 入池中 / 上分50池-核心池 / 上分50池-配置池 等 | 通用 |
| 证券交易落地营业部 | 自动/下拉 | 取资产池数据，由资产池导入或单个修改 | 通用 |
| 期货交易落地营业部 | 自动 | 周频根据子基金名称取期货经纪业务对应营业部 | 通用 |
| 上会状态 | 下拉枚举 | 见下方状态定义 | 通用 |
| 已排期金额(万) | 自动 | 通过该条上会记录创建的有效投资计划总计划金额 | 通用 |
| 待排期金额(万) | 自动 | = 金额(建仓) 或 = 调仓金额 - 已排期金额(调仓) | 通用 |

### 3.3 上会状态定义

| 状态 | 说明 | 来源 |
|------|------|------|
| **待上会** | 未豁免的初始创建状态 | 默认初始态 |
| **已上会** | 上会完成，可批量修改"待上会"→"已上会" | 批量修改 |
| **豁免上会** | 流程审批通过后的豁免标记 | 豁免流程通过 |
| **待豁免** | 已发起豁免流程但未审批结束 | 豁免流程进行中 |
| **待定** | 可批量修改"待上会"→"待定" | 批量修改 |

**状态流转规则：**
- `待上会` → `已上会` / `待定` / `豁免上会`（批量修改）
- `待上会` → `待豁免`（发起豁免流程后）
- `待豁免` → `豁免上会`（豁免流程审批通过）

### 4.6 豁免上会流程

#### 4.6.1 触发方式

1. **批量豁免**：在表格视图中勾选记录 → 点击"批量豁免" → 选择"豁免上会"
2. **新建时豁免**：录入明细时可对该条设置豁免（豁免按钮变蓝）
3. **母基金视图编辑时豁免**：编辑界面中的豁免操作

#### 4.6.2 豁免审批单弹窗

**头部区域：**
- 标签输入
- 优先级：一般 / 加急 / 特急

**母基金级别豁免：**
- 支持添加多个母基金产品
- 由**投资端**发起

**母基金+子基金级别豁免：**
- 表格形式展示已选条目（母基金、子基金、上会日期、金额）
- 支持逐条删除
- 由**商务端**发起

**审批人：** 高阳 / 章总

**备注：** 自由文本

**底部提示：**
- 流程通过后，建仓/调仓自动打豁免标签
- 自动带出最近日期的上会，可修改
- 自动带出对应日期的金额供核对，不可修改

**底部按钮：** 取消 | **确认提交**

**状态变更：**
- 发起流程后 → 该条记录变为 **"待豁免"**
- 审批通过后 → 变为 **"豁免上会"**

> 豁免流程需支持**移动端（企微）**发起和审批。

## 5. 交互规则

### 5.1 通用规则

| 规则 | 说明 |
|------|------|
| 金额格式 | 最多两位小数，有几位显示几位，无小数不显示 |
| 表格滚动 | 母基金视图内建仓/调仓表格支持横向滚动 |
| 字体统一 | 母基金视图表格统一13px字体 |
| 删除操作 | 单条删除，需确认 |

### 5.2 状态操作规则

| 操作 | 前置条件 | 结果 |
|------|----------|------|
| 批量改"已上会" | 选中"待上会"记录 | 状态→已上会 |
| 批量改"待定" | 选中"待上会"记录 | 状态→待定 |
| 批量改"豁免上会" | 选中"待上会"记录 | 提示跳转流程，确认后→待豁免 |
| 创建投资计划 | 选中"已上会"或"豁免上会"记录 | 打开排期下单弹窗 |

### 5.3 数据联动规则

- 选子基金 → 自动带出子基金管理人、策略
- 输入金额 → 自动算配置比例（=金额/总金额）
- 输入配置比例 → 自动算金额（=总金额×比例）
- 调仓后金额 = 已投金额 + 调仓金额
- 调仓后配置比例 = 调仓后金额 / (净资产 + 本次调仓总金额)
- 待排期金额 = 金额(建仓) 或 调仓金额 - 已排期金额(调仓)
- 策略比例 = 同一策略的所有配置/调仓比例之和

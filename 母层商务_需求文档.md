# 母层商务 - 产品需求文档

## 一、概述

### 1.1 功能定位
在资产管理综合-商务管理中，新增"母层商务"菜单。该菜单用于管理产品的销售情况及各项收入分配比例。

### 1.2 权限说明
- 该菜单可随时隐藏
- 修改权限同菜单查看权限

#### 一、销售情况
**表头字段**：公司归属 | 营业部 | 客户经理 | 比例(%) | 操作

| 字段 | 必填 | 类型 | 逻辑说明 |
|-----|------|------|---------|
| 公司归属 | Y | 下拉单选 | 证券/期货 |
| 营业部归属 | Y | 下拉单选带搜索 | 选择对应公司下的营业部列表 |
| 客户经理归属 | Y | 下拉单选带搜索 | 可以多选所选择的营业部下的客户经理，姓名+工号 |
| 比例(%) | Y | 数值输入 | 大于0、小于等于100，可修改 |

**标注提示**：默认分配100%（鼠标悬停显示）

**操作按钮**：
- + 添加：新增一行空白记录
- 📋 复制：复制当前section数据到其他模块（支持多选目标）

**校验规则**：
- 比例合计需等于100%
- 不等于100%时显示红色错误提示
- 等于100%时不显示提示（静默通过）

### 2.3 复制功能弹窗

#### 触发方式
点击各section的"📋 复制"按钮

#### 弹窗结构
- **源数据选择**（从）：下拉单选，自动选中当前section
- **目标选择**（复制到）：多选复选框列表
  - 包含所有7个section：销售情况、管理费、业绩报酬、认申赎费、外包费、期货户、证券户
  - 自动禁用源自身选项（置灰不可选）
  - 支持同时勾选多个目标
- **确认按钮**：执行复制操作

#### 复制逻辑
1. 读取源section的完整数据（JSON深拷贝）
2. 将数据写入所有选中的目标section
3. 自动重新渲染所有受影响的表格
4. 弹出提示："已将「XXX」复制到：YYY、ZZZ"

## 三、交互规则

### 3.1 新增产品流程
1. 点击"+ 新增"按钮
2. 打开详情弹窗（空白数据模式）
3. **每个类型默认有一行初始数据**：

| Section | 默认值 |
|---------|--------|
| 销售情况 | 证券-[空]-[空] 100% |
| 管理费 | 期货-资管部 50% |
| 业绩报酬 | 期货-[空] [空白] |
| 认申赎费 | 期货-[空] [空白] |
| 外包费 | 期货-[空] [空白] |
| 期货户 | 期货-资管部 50%<br>期货-大渡河路营业部 50% |
| 证券户 | 期货-[空] [空白] |

4. 用户可修改默认值或添加新行
5. 点击"确定"保存

### 3.2 编辑产品流程
1. 在列表页点击某行的"编辑"
2. 打开详情弹窗并加载该产品已有数据
3. 修改后点击"确定"保存

### 3.3 数据验证
- **实时校验**：输入比例时即时计算合计
- **错误提示**：比例≠100%时显示红色文字和差值
- **通过标准**：比例=100%（误差±0.01%内）且不显示成功提示
- **保存拦截**：确定时全局校验，任一section不通过则阻止保存

### 3.4 行操作
- **添加行**：点击"+ 添加"在末尾追加一行
- **删除行**：点击行末"−"按钮删除该行
- **无限制**：不限制最大行数

## 七、冲突处理说明

以下情况以**当前页面实现为准**（与原始需求截图有出入的地方）：

| 需求项 | 原始需求 | 当前实现 | 说明 |
|-------|---------|---------|------|
| 销售情况-营业部归属 | 外部展示格式中名称后面括号（securities） | 直接显示中文名称 | 简化显示 |
| 收入分配-公司归属 | 包含期货户、管理费等不同类型 | 统一为证券/期货 | 统一逻辑 |
| 校验通过提示 | 显示"✓ 比例合计 100%，校验通过" | 不显示任何提示 | 减少干扰 |
| 表格底部 | 有"全量产品"汇总行 | 已移除 | 保持简洁 |
| 新增默认值 | 未明确 | 每个section都有默认行 | 提升用户体验 |

## 八、待优化项（建议）

1. **数据持久化**：当前为前端Demo，需对接后端API进行真实数据的增删改查
2. **搜索功能**：当前为alert占位，需实现真正的全文检索
3. **批量操作**：列表页复选框已预留，可扩展批量编辑/删除功能
4. **权限控制**：根据用户角色控制字段的可见性和可编辑性
5. **历史版本**：记录每次修改的时间、操作人，支持查看变更历史
6. **导入导出**：Excel模板下载和数据批量导入
7. **审批流程**：关键数据修改需走审批流
